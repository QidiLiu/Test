# Test

## Introduction

Playground to test different algorithms.

## Projects

- `yolo11-cls-infer-openvino`: yolo11 inference code (with OpenVINO) for classification
- `yolo11-cls-train`: yolo11 training code for classification
- `yolo11-detect-infer-openvino`: yolo11 inference code (with OpenVINO) for object detection
- `yolo11-detect-train`: yolo11 training code for object detection
- `yolo11-obb-infer-openvino`: yolo11 inference code (with OpenVINO) for oriented object detection
- `yolo11-obb-train`: yolo11 training code for oriented object detection
- `yolo11-pose-infer-openvino`: yolo11 inference code (with OpenVINO) for keypoints detection
- `yolo11-pose-train`: yolo11 training code for keypoints detection
- `yolo11-seg-infer-openvino`: yolo11 inference code (with OpenVINO) for instance segmentation
- `yolo11-seg-train`: yolo11 training code for instance segmentation
